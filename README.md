# go-argparse

- A CLI parsing library inspired by AWS CLI v1
- There's already the **flag** go package and others. Why reinvent the wheel? obviously, for learning purposes :P

### run tests
```bash
$ make test
```

### run go format
```bash
$ make fmt
```
